## üì¶ How to Use

### For Telegram
1. Download the stickers you want
2. Open Telegram and go to any chat
3. Click the sticker icon and select "Add Stickers"
4. Upload the PNG files

### For Discord
1. Go to Server Settings ‚Üí Emoji
2. Click "Upload Emoji"
3. Select the sticker files (each server supports up to 50 custom emojis)

### For Direct Use
Simply right-click any "Download" link above and save the image, or copy the raw URL:

```
https://raw.githubusercontent.com/kingofthemountain-bot/joni-stickers/main/XX-name.png
```

Replace `XX-name` with the sticker number and name (e.g., `01-happy`, `14-coding`, etc.)
